# Astro Personal Site Starter

Minimal, high‑key Astro site for blog (with tags), résumé, physics research, and photography.

## Setup

```bash
# 1) Unzip this into a folder, cd into it, then:
npm install

# 2) Start dev server
npm run dev   # http://localhost:4321

# Optional: initialize Biome
npx @biomejs/biome init
npm run lint
```
